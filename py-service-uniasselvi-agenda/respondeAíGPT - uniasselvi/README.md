# respondeAiGPT - uniasselvi

Frontend web simples para interacao com fluxos ligados a ChatGPT e Uniasselvi.

## Arquivos principais

- `index.html`: interface principal.
- `main.js`: logica da pagina.
- `style.css` e `popup.css`: estilos.

## Uso

Abra `index.html` no navegador ou adapte para servir o frontend por um servidor estatico.
