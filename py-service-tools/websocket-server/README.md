# websocket-server

Servidor WebSocket assincro para comunicacao em tempo real, acompanhado de uma pagina cliente de apoio.

## Arquivos principais

- `servidor.py`: servidor Python.
- `index.html`: cliente simples.
- `server-websocket-client/`: materiais complementares.

## Uso

Revise host, porta e contratos de mensagem antes de expor este servico em rede.
