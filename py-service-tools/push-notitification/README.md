# 🗂️ Push Notification — exemplo com Plyer

Exemplo mínimo que demonstra uma notificação desktop usando a biblioteca `plyer`.

---

## ✅ Funcionalidades

- 🔔 Exibe notificação desktop com título e mensagem

---

## ⚙️ Requisitos

- Python 3.7+
- Dependência:

```bash
pip install plyer
```

## 🚀 Como usar

1. Instale a dependência:

```powershell
pip install plyer
```

2.Execute:

```powershell
python main.py
```

---

## 🧩 Observações

- Comportamento pode variar entre Windows/macOS/Linux. Em ambientes sem suporte a notificações, capture exceções.
