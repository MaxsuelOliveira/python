# py-service-opencv-faces

Diretorio agrupador com experimentos de visao computacional usando OpenCV em diferentes versoes.

## Versoes

- `v0/`: primeira iteracao com deteccao de rostos, gravacao de video e captura de tela.
- `v1/`: evolucao da base com ramificacoes para analise de placas, rostos e reconhecimento de texto.

## Uso

Escolha a versao desejada e siga o `README.md` interno.
