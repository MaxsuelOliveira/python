
# 🗂️ My Home — interface web estática

Front-end estático com páginas HTML/JS/CSS para exibir informações locais (ex: tempo, clima, webcam). Ideal para deploy em GitHub Pages como site estático.

---

## ✅ Funcionalidades

- 🖥️ Páginas HTML prontas (`index.html`, `main.html`)
- 🎛️ Scripts JS para interações simples

---

## ⚙️ Requisitos

- Navegador moderno (Chrome/Firefox/Edge)

## 🚀 Como usar

1. Abrir `index.html` localmente ou servir com um servidor estático:

```powershell
python -m http.server 8000
```

2.Abrir `http://localhost:8000` no navegador.

---

## 🧩 Observações

- Para dados em tempo real, implemente um backend (WebSocket ou API) e aponte os scripts JS para ele.
