# py-api-alexia

Agrupa tres subprojetos relacionados a automacao, interface local e controle de temperatura.

## Subprojetos

- `api-controladorar-fita-de-led-bluetooh/`: controle de fita LED via Bluetooth.
- `api-my-desktop/`: interface web/local com frontend e servidor Python.
- `service-controle-temperatura/`: automacao de temperatura com scripts e funcao Lambda.

## Como usar

Entre no diretorio do subprojeto desejado e siga o `README.md` local quando existir.

## Observacao

Este diretorio funciona como agrupador. Dependencias, variaveis de ambiente e modo de execucao variam por subprojeto.
