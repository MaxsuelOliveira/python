# py-api-buscador-de-dados

Colecao de APIs, utilitarios de consulta e versoes de um mesmo projeto de busca de dados.

## Subprojetos

- `consultar-placas/`: consulta de placa de veiculo.
- `consultar-serasa/`: consulta de dados relacionados a Serasa.
- `consultar-veiculo/`: consulta de dados de veiculo.
- `main/`: aplicacao principal com `app.py` e `db.py`.
- `v1.0/`: versao estruturada com `src/`, `docs/` e banco local.
- `v1.1/`: versao simplificada com scripts de consulta.

## Uso

Cada pasta possui contexto proprio. Use este diretorio como indice para localizar a versao ou consulta desejada.
