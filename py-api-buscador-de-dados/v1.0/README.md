# v1.0

Versao estruturada da API de busca de dados, com organizacao por camadas em `src/`, documentacao em `docs/` e bancos locais em `data/`.

## Componentes principais

- `main.py`: ponto de entrada.
- `src/`: bot, controllers, database, models, services e views.
- `docs/`: notas de implantacao e endurecimento.
- `data/`: arquivos SQLite locais.

## Observacao

Este diretorio parece representar uma versao mais completa e organizada da aplicacao.
